Aleatory
========

Generating random stuff, by chance ...
... in Clojure(script)


Note that it's an experimental repository, nothing much usable for now


## Unit tests

To launch the unit tests:

Clojure (JVM runtime):

```
lein test
```


Clojurescript (default runtime nodejs):

```
lein doo
```
